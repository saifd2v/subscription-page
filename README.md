# Subscription Page

This is a **fully responsive subscription page** built purely with **HTML and CSS**.  
It showcases different subscription plans with their features, prices, and interactive buttons.

## Features

- Clean and modern design using **CSS Flexbox**.
- Multiple subscription plans: Free, Basic, Advanced, and Pro.
- Interactive buttons with smooth hover effects.
- Responsive layout that adapts to **desktop, tablet, and mobile devices**.
- Organized display of features for each plan using icons from **Font Awesome**.
- Highlighted recommended plan with a special badge.

## Technologies Used

- HTML5
- CSS3 (Flexbox, Media Queries)
- Font Awesome for icons
- Google Fonts (Tajawal)

## Responsive Design

The page layout adjusts automatically to different screen sizes:

- **Desktop**: Plans are displayed in a row with vertical dividers.  
- **Tablet**: Plans wrap into multiple rows for better readability.  
- **Mobile**: Plans stack vertically with proper spacing and scaling.

## Notes

- This project does **not include JavaScript**; it focuses purely on layout and styling.
- It is intended for **learning and practice purposes** in front-end development.
